# README - Site do Podcast "Vozes da Amazônia"

Este documento serve como guia principal para todos os arquivos entregues para o site do podcast "Vozes da Amazônia".

## Conteúdo da Entrega

### 1. Protótipo HTML Completo
- **Arquivos HTML**: Páginas completas do site (index.html, sobre.html, episodios.html, contato.html, eventos.html)
- **CSS**: Estilos responsivos e otimizados (pasta css)
- **JavaScript**: Funcionalidades interativas (pasta js)
- **Imagens**: Arquivos de imagem utilizados no site (pasta images)

### 2. Wireframes
- Wireframes detalhados para todas as páginas principais do site

### 3. Documentação
- **guia_wordpress_vozes_amazonia.md**: Instruções detalhadas para configuração do WordPress
- **implementacao_design_wordpress.md**: Guia passo a passo para implementar o design no WordPress
- **guia_manutencao_site_vozes_amazonia.md**: Instruções para manutenção regular do site
- **validacao_site_vozes_amazonia.md**: Documento de validação confirmando que todos os requisitos foram atendidos

### 4. Arquivos Originais
- Logo do podcast (vozes.jpg)
- Captura de tela do perfil do Instagram (image.png)

## Como Utilizar Esta Entrega

### Para Apresentação em Reunião
1. Abra os arquivos HTML em um navegador para demonstrar o design e funcionalidades do site
2. Utilize os wireframes para explicar a estrutura e organização do conteúdo
3. Consulte o documento de validação para confirmar que todos os requisitos foram atendidos

### Para Implementação do Site
1. Siga as instruções no **guia_wordpress_vozes_amazonia.md** para configurar o WordPress gratuitamente
2. Utilize o **implementacao_design_wordpress.md** para recriar o design no WordPress
3. Consulte o **guia_manutencao_site_vozes_amazonia.md** para manter o site atualizado e seguro

## Requisitos Atendidos

- ✅ Site profissional para o podcast
- ✅ Solução totalmente gratuita (WordPress com temas e plugins gratuitos)
- ✅ Design seguindo a identidade visual do Instagram
- ✅ Layout inspirado no site "Não Inviabilize"
- ✅ Seção "Conheça os entrevistados de cada episódio"
- ✅ Incorporação de informações sobre trabalhos acadêmicos e TCCs
- ✅ Destaque para a futura 3ª temporada "Vozes da Amazônia: Ciência e Saberes em Diálogo"

## Próximos Passos Recomendados

1. Revisar o protótipo HTML e fornecer feedback sobre quaisquer ajustes desejados
2. Seguir o guia de configuração do WordPress para implementar o site
3. Adicionar conteúdo real (episódios, informações de entrevistados, eventos)
4. Configurar o domínio gratuito e a hospedagem conforme as instruções
5. Implementar o plano de manutenção regular

## Suporte

Para quaisquer dúvidas ou assistência adicional, consulte a documentação oficial do WordPress ou entre em contato com um desenvolvedor especializado.
